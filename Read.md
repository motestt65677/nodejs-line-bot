# nodejs-line-bot 
##testing environment
- Create .env file
- Install node js. (see https://nodejs.org/en/download/)
- Use Ngrok to get a https url. 
- Set webhook on https://developers.line.biz/, make sure the route is https. (ex. https://dc7112cf3391.ngrok.io/linewebhook)
- run ```node app.js```
